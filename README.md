# terminal games!
terminal implementations of tic tac toe, hangman, and checkers (more to come) written in perl and python using the curses library.

## dependencies
 - [perl v5.2](https://www.perl.org/)
 - [curses module](http://search.cpan.org/~giraffed/Curses-1.36/Curses.pm)
 - [Time::HiRes module](http://perldoc.perl.org/Time/HiRes.html)
 - [python3](https://www.python.org/)
 - [numpy](http://www.numpy.org/)
